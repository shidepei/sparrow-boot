# sparrow-boot
**项目结构** 
```
sparrow-boot
├─sql  项目SQL语句
│
├─boot-common 公共模块
│ 
├─boot-cache 缓存模块
│ 
├─boot-datasource 数据源
│ 
├─boot-admin 后台管理

```

**技术选型：** 
- 核心框架：Spring Boot 2.1.3
- 安全框架：Apache Shiro 1.4
- 常用工具包：Hutool 4.5.1
- 持久层框架：mybatis-plus 3.1.0
- 接口文档： swagger 2.7.0
- 页面交互：Vue2.x + element-UI
- JDK版本：1.8
- mysql版本：5.6
- redis版本：3.2.1
- 开发工具：IntelliJ IDEA 2018.3.4 x64、Navicat Premium、RedisDesktopManager
